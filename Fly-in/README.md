# Fly-in

This project is present here: https://github.com/Diogo-Serra/Fly-in

---

A drone fleet routing system that efficiently schedules multiple drones from a starting hub to a destination while navigating a network of zones under strict constraints. The system reads a graph-based network definition, applies pathfinding algorithms to distribute drones across optimal routes, respects zone and connection capacity limits, and accounts for different movement costs based on zone types. The goal is to move all drones to the end zone in the minimum number of simulation turns while providing visual feedback of the process.
